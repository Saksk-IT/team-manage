质保名单判定

控制前台质保服务的邮箱名单判定模式，并维护命中 / 未命中时随机展示且按邮箱锁定的富文本模板。

前台质保邮箱名单判定

开启后，前台质保服务输入邮箱只判断该邮箱是否存在于质保邮箱列表，不再查询订单、刷新 Team 状态或提交质保；首次查询会随机锁定一个模板，后续同邮箱固定展示该模板。

至少保留 1 个模板；邮箱首次命中后会随机锁定其中一个。

至少保留 1 个模板；邮箱首次未命中后会随机锁定其中一个。

支持常用富文本：加粗、链接、列表、段落、图片；可直接粘贴图片或点击“图片”上传，单张不超过 5MB。

开启名单判定后，邮箱命中质保列表时，系统会按该邮箱剩余天数（向上取整）调用 Sub2API Admin API 生成一个未使用的订阅兑换码；同一邮箱后续查询复用同一兑换码。
