Open Claw

开始前请先完成：回到 Codex 总教程第二章 创建 API Key。Open Claw 的 OpenAI-compatible 地址通常填写 https://api.sakms.top/v1。

Open Claw 支持两种部署方式：腾讯云在线配置（推荐新手）和本地配置。

1. 腾讯云在线配置

适用场景：你已经在腾讯云开通了“龙虾服务器”，希望直接通过云端面板接入。

  1. 登录腾讯云，进入你的龙虾服务器控制台。
  2. 点击“应用管理” → 找到“模型”模块。
  3. 在下拉菜单中选择“自定义模型” → “JSON 输入”。
  4. 把下方 JSON 粘贴进去，替换 API Key 后点击“添加并应用”。
{
  "provider": "openai",
  "base_url": "https://api.sakms.top/v1",
  "api": "openai-completions",
  "api_key": "填写你的 API 密钥",
  "model": {
    "id": "gpt-5.5",
    "name": "GPT-5.5"
  }
}

注意：api_key 字段务必替换为自己的 API Key，不要直接提交带有占位符的 JSON。

2. 本地配置（Windows / macOS / Linux）

系统配置目录打开方式
Windows%USERPROFILE%\.openclaw按 Win + R，输入 %USERPROFILE%\.openclaw 并回车。
macOS~/.openclaw终端执行 open ~/.openclaw。
Linux~/.openclaw终端执行 xdg-open ~/.openclaw 或直接进入。

进入上述目录，编辑或新建 openclaw.json，写入：

{
  "baseUrl": "https://api.sakms.top/v1",
  "api": "openai-responses",
  "apiKey": "填写你的 API 密钥"
}

提示：Open Claw 本地配置通常使用 openai-responses，与腾讯云端示例里的 openai-completions 不同，不要混用。

3. 快速检查

  • base_url / baseUrl 是否为 https://api.sakms.top/v1。
  • api_key / apiKey 是否已替换为自己的真实 API Key。
  • 云端配置使用 openai-completions，本地配置使用 openai-responses，不要混用。
